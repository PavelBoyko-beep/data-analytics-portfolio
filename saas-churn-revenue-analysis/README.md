# SaaS Churn & Revenue Analysis

## Project Overview

This project analyzes a fictional SaaS company with subscription-based revenue.  
The goal is to understand customer churn, recurring revenue, customer segments, and retention patterns.

## Business Problem

A SaaS company wants to understand why customers cancel their subscriptions, which plans generate the most revenue, and what actions can improve retention and protect MRR.

## Tools

- PostgreSQL / SQL
- Python / pandas
- Power BI
- GitHub

## Project Structure

- data/raw — original dataset files
- data/processed — cleaned dataset files
- sql — SQL scripts for table creation, import, quality checks, and analysis
- notebooks — Python notebook for basic data cleaning and EDA
- dashboard — Power BI dashboard and screenshots
- docs — business questions, data dictionary, and insights
